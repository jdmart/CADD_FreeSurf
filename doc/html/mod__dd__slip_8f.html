module  mod_dd_slip
 — This module contains the entire DD methodology — Generate slip planes — Generate sources to a given density — Generate obstacles to a given spacing lobs — Calculate the P-K force (direct or multipole) – currently only direct — Calculate the displacement at any location — Calculate the stress at any given point — Handling of pinned dislocations
